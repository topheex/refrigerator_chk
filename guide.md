# 🚀 5살도 할 수 있는 Render.com 배포 가이드

내 냉장고 관리 앱을 인터넷에 공개해 볼까요? 
이 가이드는 초등학생도 따라 할 수 있게 아주 쉽게 설명했어요!

---

## 1. 회원가입 및 연동
1. **[Render.com](https://render.com/)** 사이트에 접속하세요.
2. **'Get Started'** 버튼을 누르고, 우리 코드가 저장되어 있는 **'GitHub'** 계정으로 로그인하세요. (계정이 없으면 가입하기!)
3. 로그인이 완료되면 오른쪽 상단에 **'New +'** 라는 버튼이 있어요. 그걸 누르고 **'Web Service'**를 선택하세요.

---

## 2. 코드 가져오기 (연동)
1. 화면에 **'Connect a repository'** 라는 글자가 보일 거예요. 거기서 우리가 만든 레포지토리인 `refrigerator_chk` 옆의 **'Connect'** 버튼을 누르세요.
   * *만약 안 보이면 'Configure account'를 눌러서 GitHub 권한을 허용해 줘야 해요!*

---

## 3. 설정 (중요!)
이제 컴퓨터에게 어떻게 빌드하고 실행할지 알려줄 거예요.

* **Name:** 아무 이름이나 지어주세요. (예: `my-refrigerator-app`)
* **Root Directory:** `server` 라고 적으세요. (이게 중요해요!)
* **Runtime:** `Node`가 선택되어 있는지 확인하세요.
* **Build Command:** `npm run build` 라고 적으세요.
* **Start Command:** `npm start` 라고 적으세요.

---

## 4. 마지막 단계
1. 페이지 아래로 쭉 내려가서 **'Create Web Service'** 라는 버튼을 누르세요.
2. 그러면 화면에 검은색 창이 나타나고 글자가 막 올라갈 거예요. 이게 코드를 정리하고 준비하는 과정이에요!
3. 몇 분 정도 기다리면, 오른쪽 위에 **'Your service is live'** 라는 글자가 나타나요.
4. 그 아래에 있는 `https://...` 링크를 클릭해보세요!

**우와! 내 냉장고 앱이 인터넷 세상에 나왔어요! 🎉**

---

### 💡 팁

* 앱을 수정해서 GitHub에 올리면, Render가 자동으로 다시 빌드해서 새 버전으로 바꿔줘요. 참 똑똑하죠?
