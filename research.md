# Refrigerator Checker (refrigerator_chk)

이 프로젝트는 냉장고 속 식재료들을 관리하는 Full-stack 애플리케이션입니다.

## 기술 스택

### Frontend (`/client`)
- **Framework**: React
- **Language**: TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS

### Backend (`/server`)
- **Runtime**: Node.js
- **Framework**: Express
- **Database**: SQLite (via `better-sqlite3`)

## 프로젝트 구조

- `client/`: React 프론트엔드 프로젝트
- `server/`: Express 백엔드 서버 및 SQLite 데이터베이스
